# Landing-Page
## Description  
Responsive landing page, fully made with CSS3 Flexbox layout model.  
It is made for the purpose of training with Flexbox layout model, and understanding the principles of responsiveness.  
## Technologies  
Tech-Stack:  
<ul>
  <li>HTML5</li>
  <li>CSS3</li>
</ul>
